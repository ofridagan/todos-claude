# todos-claude

Claude-powered todos management system with multiple spaces.

## Install

```bash
git clone https://github.com/ofridagan/todos-claude.git ~/dev/todos-claude
cd ~/dev/todos-claude
claude
```

**Or add this alias to your shell:**

```bash
alias todo='(cd ~/dev/todos-claude && claude)'
```

Then just use `todo` from anywhere.

## Get Started

Run `/help` to see the quick start guide and all available commands.
